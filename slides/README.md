# Slides — AI-First Software Development

Slidev-Präsentation mit Speaker Notes.

## Setup

```bash
pnpm install
pnpm run dev
```

→ http://localhost:3030

## Presenter-Modus

Drücke `s` in der Präsentation für den Presenter-View mit Speaker Notes.

## Export

```bash
pnpm run export-pdf    # PDF
pnpm run export-pptx   # PowerPoint
```

Für Export wird Playwright benötigt:
```bash
npx playwright install chromium
```
